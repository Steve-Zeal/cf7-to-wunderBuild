# CF7 to Wunderbuild CRM Connector

This WordPress plugin connects Contact Form 7 submissions to Wunderbuild CRM, allowing you to automatically send form submissions to your Wunderbuild CRM account.

## Requirements

- WordPress 5.0 or higher
- Contact Form 7 plugin installed and activated
- A Wunderbuild CRM account with API access

## Installation

1. Download the plugin files
2. Upload the `cf7-to-wunderbuild` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Settings > CF7 to Wunderbuild to configure your API settings

## Configuration

1. Get your API key from your Wunderbuild CRM account
2. Go to the WordPress admin panel > Settings > CF7 to Wunderbuild
3. Enter your API key
4. Select which Contact Form 7 fields you want to send to Wunderbuild
5. Save your settings

## Usage

Once configured, the plugin will automatically send form submissions to Wunderbuild CRM whenever a Contact Form 7 form is submitted successfully.
